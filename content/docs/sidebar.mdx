# Sidebar

A VisionOS-style sidebar component that provides navigation with collapsible functionality and active state detection.

## Features

- **VisionOS Design**: Follows the frosted glass design language with material effects
- **Collapsible**: Can be collapsed to show only icons or expanded to show full text
- **Active State Detection**: Automatically highlights the active menu item based on current route
- **FlashList Integration**: Uses FlashList for efficient rendering of menu items
- **Flexible Layout**: Supports left/right positioning and custom headers/footers
- **Badge Support**: Menu items can display badges for notifications or counts

## Basic Usage

```tsx
import { Sidebar, type SidebarItemProps } from '@/components/core/v2/sidebar'
import { IconHome, IconUsers, IconSettings } from '@tabler/icons-react'

const sidebarItems: SidebarItemProps[] = [
  { name: 'Home', href: '/', icon: <IconHome />, title: 'Home' },
  { name: 'People', href: '/people', icon: <IconUsers />, title: 'People', badge: '3' },
  { name: 'Settings', href: '/settings', icon: <IconSettings />, title: 'Settings' },
]

export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <Sidebar items={sidebarItems}>
      {children}
    </Sidebar>
  )
}
```

## Props

### SidebarProps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Content to render in the main area |
| `items` | `SidebarItemProps[]` | - | Array of sidebar menu items |
| `position` | `'left' \| 'right'` | `'left'` | Position of the sidebar |
| `floating` | `boolean` | `true` | Whether to apply floating shadow effect |
| `collapsible` | `boolean` | `true` | Whether the sidebar can be collapsed |
| `header` | `React.ReactNode \| (() => React.ReactNode)` | - | Optional header content |
| `footer` | `React.ReactNode \| (() => React.ReactNode)` | - | Optional footer content |
| `className` | `string` | - | Additional CSS classes for the container |
| `contentClassName` | `string` | - | Additional CSS classes for the content area |

### SidebarItemProps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | - | Unique identifier for the menu item |
| `href` | `string` | - | Navigation URL |
| `icon` | `React.ReactNode` | - | Icon to display |
| `title` | `string` | - | Display text (falls back to `name`) |
| `badge` | `string \| number` | - | Optional badge to display |
| `disabled` | `boolean` | `false` | Whether the item is disabled |

## Active State Detection

The sidebar automatically detects the active menu item based on the current route:

1. **Exact Match**: First tries to find an item where `href` exactly matches the current pathname
2. **Subpath Match**: If no exact match, finds an item where the current pathname starts with the item's `href`

For example:
- If current route is `/people`, it will match an item with `href: '/people'`
- If current route is `/people/123`, it will also match an item with `href: '/people'`

## Advanced Usage

### Custom Header and Footer

```tsx
<Sidebar 
  items={sidebarItems}
  header={() => (
    <div className="px-2 py-1">
      <h2 className="font-semibold text-lg text-white/90">Navigation</h2>
    </div>
  )}
  footer={() => (
    <div className="px-2 py-1">
      <p className="text-sm text-white/60">Version 1.0.0</p>
    </div>
  )}
>
  {children}
</Sidebar>
```

### Non-collapsible Sidebar

```tsx
<Sidebar 
  items={sidebarItems}
  collapsible={false}
>
  {children}
</Sidebar>
```

### Right-positioned Sidebar

```tsx
<Sidebar 
  items={sidebarItems}
  position="right"
>
  {children}
</Sidebar>
```

## Design System Integration

The Sidebar component integrates seamlessly with other VisionOS v2 components:

- Uses `Material` component for the frosted glass effect
- Integrates with `FlashList` for efficient rendering
- Uses `Button` component for menu items
- Follows the same design tokens and spacing as other v2 components
